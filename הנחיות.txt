המטרה הנוכחית

המטרה הנוכחית היא להגיע לכיסוי מלא של כל החברות ברשימה.

המערכת צריכה להיכנס בפועל לדפי המשרות האמיתיים בכל חברה, לקרוא את תוכן המשרה, ולסנן רק משרות רלוונטיות לפי תחום ומיקום.

לא מספיק:

למצוא לינקים בלבד.
להציג Job-like URLs בלבד.
להסתמך רק על כותרת הקישור.
להחזיר תוצאה בלי לוודא שנקרא דף משרה אמיתי.

מכל משרה רלוונטית יש לחלץ לפחות:

Company
Title
URL
Matched keyword
Matched location, אם זמין

סטטוס נוכחי

הפרויקט כבר עבר ל-GitHub.
main.py מריץ סריקה לכל החברות או לחברה אחת לפי ארגומנט.
ScannerFactory בוחר סורק לפי platform.
scanner.py הוא עדיין מנוע הסריקה המרכזי בפועל.
WorkdayScanner קיים אך כרגע עוטף את scan_company.
DEBUG=True כרגע כדי להבין למה חברות לא מחזירות תוצאות.

השלב הנוכחי הוא לא להוסיף פיצ׳רים חדשים, אלא לייצב כיסוי אמיתי של חברות ופלטפורמות.

Definition of Done לחברה

חברה נחשבת ✅ רק אם:

הסורק פותח את אתר הקריירה שלה.
הסורק מגיע לדף משרה אמיתי.
הסורק קורא את תוכן המשרה בפועל.
הסורק מחלץ Title ו-URL.
הסורק מזהה Location כאשר קיים.
הסורק מזהה Keyword רלוונטי כאשר קיים.
המשרה מופיעה תחת Relevant jobs found.
אין Exception בהרצה נקודתית של החברה.

Regression checklist

אחרי כל שינוי משמעותי יש להריץ לפחות:

python main.py NVIDIA
python main.py Apple
python main.py Intel
python main.py Microsoft

ולאחר מכן, כאשר השינוי נראה תקין:

python main.py

אם חברה שכבר עבדה נשברת, קודם מתקנים את הרגרסיה ורק אחר כך ממשיכים.

Definition of Success ל-v0.1

v0.1 תיחשב יציבה כאשר:

כל החברות המרכזיות ברשימה הן ✅ או שיש סיבה מתועדת וברורה למה לא.
אין קריסות בהרצה מלאה.
כל משרה שמוחזרת מגיעה מדף משרה אמיתי.
אין False Positives ידועים.
הפלט כולל Company, Title, URL, Matched keyword ו-Matched location כאשר זמין.
